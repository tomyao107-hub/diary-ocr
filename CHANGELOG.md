# Changelog

## 1.0.1 — 2026-07-18

- 修复：合并输出后错误清除项目 session，导致自定义顺序与重开恢复失效。
- 修复：校对台关闭时把项目 `output/` 写进全局配置，污染默认输出目录。
- 修复：返回项目列表时用首页旧配置覆盖编辑器内已保存的 API 设置。
- 修复：进度保存/清除对话框显示错误的全局 session 路径。
- 修复：OCR API 返回 `content=None` 时崩溃。
- 增强：会话加载跳过非法/越界路径，不再整文件失效。
- 增强：文件夹导入单张失败可跳过并继续；PDF 空页/加密/复制失败报错更清晰。
- 增强：Windows 保留名（CON/NUL 等）项目目录自动改名；页数只统计图片。
- 增强：配置与文本写入统一原子写并清理临时文件。

## 1.0.0 — 2026-07-18

- 新增项目首页与自包含项目目录。
- 新增项目扫描、新建、隐藏及安全删除。
- 图片导入改为复制进项目，并支持递归文件夹导入。
- 新增 PDF 页码范围、DPI、逐页渲染、进度和取消。
- 会话改为项目级存储，使用可迁移的相对页面路径。
- OCR 与合并输出固定写入当前项目的 `output/`。
- 新增 `diary_ocr_app.py` 稳定入口与模块化 `diary_ocr` 包。
